# 🗂️ Task Nest

**Task Nest** is a simple and interactive task management application built using **React.js**.
The idea behind the name comes from a bird’s nest — where everything is organized in one place.
Similarly, this app helps manage tasks with different **priorities and categories in a single organized space**.

---

## 🚀 Features

* Add, edit, and delete tasks
* Mark tasks as complete / undo
* Set task priorities (High, Medium, Low)
* Organize tasks by categories (Work, Personal, General)
* Sidebar filters with dynamic task counts
* Dashboard with total tasks, completed tasks, and progress
* Data persistence using Local Storage
* Clean and responsive UI

---

## 🛠️ Tech Stack

* React.js
* JavaScript (ES6)
* CSS3

---

## ⭐ Key Features

* Component-based architecture using React
* State management using React Hooks
* Dynamic filtering and real-time updates
* User-friendly and intuitive interface
